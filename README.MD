
# MoreInfinityCell

**版本：v0.0.2**

## 简介
本项目为 KubeJS/GTLeisure 提供了更多“无限元件”相关配方，适用于 Mekanism 控制电路、各电压等级的通用电路板等。

## 特性
- 新增 Mek 控制电路无限元件配方
- 新增 ULV 至 MAX 电压的通用电路板无限元件配方
- 可拆解无尽反物质燃料棒无限元件，获得反物质、湮灭约束器、液态无尽无限元件
- 通用电路板无限元件添加了 NBT 描述文本
- 其他更新特性详见[CHANGELOG.MD](https://github.com/Misaka2592/More-Infinity-Cell-for-GTLeisure/blob/main/CHANGELOG.MD)

> ⚠️ 本项目不保证配方平衡，仅根据电压等级选取相应材料。

## 依赖项
### MOD
- GTL Additions（需使用 fork 分支：[Dragonators/GTLAdditions](https://github.com/Dragonators/GTLAdditions)）
### KubeJS
- 产线撕裂者（thetornproductionline)
- （可选）产线爆破（cxbp）
## 安装方法
1. 将本项目文件分别放入 `kubejs` 目录下的 `client_scripts` 和 `server_scripts` 文件夹中，或是整体放入`kubejs`目录中。
2. 重启游戏或服务器以完成配方重载与JEI注册。

## 使用说明
- 直接在游戏内查看 JEI 配方即可。
- 由于暂时没有添加 IV 至 MAX 电路主机无限元件的配方，建议配合其他拥有获取该无限元件的私货使用
